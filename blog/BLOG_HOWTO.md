# ブログ記事の追加方法

## 📝 記事を追加する手順

### 1. `blog/posts.json` を開く

### 2. 先頭（`[` の直後）に以下をコピペして内容を書き換える

```json
{
  "slug":    "YYYYMMDD-記事キーワード",
  "title":   "記事タイトル",
  "date":    "YYYY-MM-DD",
  "excerpt": "一覧に表示される概要文（60文字くらいが目安）",
  "image":   "./images/ブログ画像ファイル名.jpg"
},
```

#### 入力例
```json
{
  "slug":    "20260401-new-drink",
  "title":   "新ドリンク「春サワー」が登場！",
  "date":    "2026-04-01",
  "excerpt": "春らしいピンク色のサワーが新登場。旬のいちごを使ったさわやかな一杯です。",
  "image":   "./images/blog-spring-sour.jpg"
},
```

---

## 📌 各項目の説明

| 項目 | 内容 | 例 |
|------|------|----|
| `slug` | 記事のURL用ID。日付＋キーワードで。**英数字・ハイフンのみ** | `20260401-new-drink` |
| `title` | 記事タイトル | `新ドリンクが登場！` |
| `date` | 公開日。**YYYY-MM-DD形式** | `2026-04-01` |
| `excerpt` | 一覧に出る概要文 | `60文字くらいが目安` |
| `image` | サムネイル画像。`images/`フォルダに入れてパスを書く | `./images/blog-xxx.jpg` |

---

## ⚠️ よくあるミスと確認ポイント

- 最後の記事の `}` の後に **カンマ（,）をつけない**
- `slug` に日本語・スペース・記号は使わない
- `date` は必ず `"2026-04-01"` の形式（ハイフン区切り）
- 画像ファイルは必ず `blog/images/` に入れておく

---

## 🗂️ ファイル構成

```
nishiume-site/
└── blog/
    ├── posts.json        ← ここに記事を追記するだけ！
    └── images/
        ├── blog-spring-menu.jpg
        ├── blog-hanami.jpg
        └── ...（ブログ用画像はここに入れる）
```

---

## ✅ 追記後のposts.jsonのイメージ

```json
[
  {
    "slug":    "20260401-new-drink",   ← 新しい記事（先頭に追加）
    "title":   "新ドリンクが登場！",
    "date":    "2026-04-01",
    "excerpt": "春らしいサワーが新登場。",
    "image":   "./images/blog-spring-sour.jpg"
  },
  {
    "slug":    "20260306-spring-menu",  ← 既存の記事
    ...
  }
]
```

新しい記事を**先頭に追加**していくのがおすすめです（自動で日付順に並び替えるので順番は何でもOK）。
